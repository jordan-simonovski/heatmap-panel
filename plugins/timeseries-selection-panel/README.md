# Timeseries Selection Panel

Grafana panel plugin providing a timeseries line chart with horizontal brush selection for comparison analysis.

## Features

- Canvas-based line chart rendering
- Horizontal brush selection (drag to select a time range)
- Context menu: Analyse Selection, Zoom to time range
- Publishes selection events compatible with the shared-comparison package
- Configurable line color, fill opacity, threshold line, and Y-axis label
